#iBurn-iOS

## Installation

 * Install Cocoapods
 
 * Clone the iBurn repo
 
 * cd to the iBurn repo
 
 * pod install

 * open iBurn.xcworkspace

 * add libProj4.a to the MapBox pod, because of a bug
